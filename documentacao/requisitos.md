# 📋 Requisitos Funcionais do Sistema

| ID    | Tipo | Nome                            | Descrição                                                  | Prioridade | Critério de aceite                                          |
| ----- | ---- | ------------------------------- | ---------------------------------------------------------- | ---------- | ----------------------------------------------------------- |
| RF-01 | RF   | Cadastro de usuário responsável | O sistema deve permitir o cadastro de pais ou responsáveis | Alta       | Usuário responsável pode se cadastrar com dados válidos     |
| RF-02 | RF   | Cadastro de usuário criança     | Cadastro de criança atrelado a uma conta de responsável    | Alta       | Usuário criança pode se cadastrar com dados válidos         |
| RF-03 | RF   | Autenticação do usuário         | O sistema deve permitir login                              | Alta       | Usuário cadastrado pode fazer login com dados válidos       |
| RF-04 | RF   | Atividades progressivas         | O sistema deve apresentar atividades lúdicas progressivas  | Alta       | Sistema apresenta atividades conforme o nível de ansiedade  |
| RF-05 | RF   | Registro de ansiedade           | Registrar ansiedade antes e depois das atividades          | Alta       | Usuário registra ansiedade antes e depois de cada atividade |
| RF-06 | RF   | Histórico de atividades         | Permitir acesso ao histórico de atividades                 | Média      | Responsável consulta histórico da criança                   |
| RF-07 | RF   | Diário emocional                | Criação de registros descritivos                           | Alta       | Registros são salvos e podem ser consultados                |
| RF-08 | RF   | Busca no diário                 | Buscar registros por data, título ou palavras-chave        | Média      | Usuário consegue buscar registros corretamente              |
| RF-09 | RF   | Identificação de padrões        | Identificar padrões de ansiedade                           | Média      | Sistema apresenta padrões com base nos registros            |
| RF-10 | RF   | Visualização de evolução        | Gerar gráficos de evolução                                 | Alta       | Gráficos são exibidos corretamente                          |
| RF-11 | RF   | Listar crianças                 | Listar crianças vinculadas ao responsável                  | Alta       | Apenas crianças vinculadas são exibidas                     |
| RF-12 | RF   | Gerenciamento de contas         | Editar ou excluir cadastros                                | Alta       | Alterações salvas corretamente e exclusões sem erro         |
| RF-13 | RF   | Notificações de atividades      | Enviar lembretes de atividades                             | Média      | Usuário recebe notificações conforme configuração           |
| RF-14 | RF   | Notificações de registro        | Lembretes para registros diários                           | Média      | Usuário recebe lembretes corretamente                       |
| RF-15 | RF   | Personalizar notificações       | Configurar horário e conteúdo das notificações             | Baixa      | Configurações são aplicadas corretamente                    |
| RF-16 | RF   | Conclusão de atividade          | Marcar atividades como concluídas                          | Alta       | Atividade aparece como concluída no histórico               |
| RF-17 | RF   | Controle de permissões          | Diferenciar permissões entre responsável e criança         | Alta       | Permissões respeitadas corretamente                         |
