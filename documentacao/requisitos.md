# Requisitos Funcionais

| ID  |Tipo| Nome |
|-----|----------|

| RF02 | O sistema deve permitir login |
| RF03 | O sistema deve apresentar atividades lúdicas progressivas |
| RF04 | O sistema deve registrar ansiedade antes e depois de cada atividade realizada |    
| RF05 | O sistema deve oferecer técnicas de autoregulação |
| RF06 | O sistema deve armazenar registros no diário emocional |
| RF07 | O sistema deve identificar padrões de ansiedade |
| RF08 | O sistema deve gerar gráficos de evolução |
