# Requisitos Funcionais

| ID  | Requisito |
|-----|----------|
| RF01 | O sistema deve permitir o cadastro de usuários |
| RF02 | O sistema deve permitir login |
| RF03 | O sistema deve apresentar desafios progressivos |
| RF04 | O sistema deve registrar ansiedade antes e depois dos desafios |
| RF05 | O sistema deve oferecer técnicas de respiração |
| RF06 | O sistema deve armazenar registros no diário emocional |
| RF07 | O sistema deve identificar padrões de ansiedade |
| RF08 | O sistema deve gerar gráficos de evolução |
