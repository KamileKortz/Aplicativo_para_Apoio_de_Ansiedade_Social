# Caso de Uso – RF-12: Gerenciamento de contas

## Descrição
Editar ou excluir cadastros.

## Atores
- Responsável.

## Pré-condições
- Usuário autenticado.

## Fluxo principal
1. O ator acessa as configurações de conta.
2. O ator altera os dados ou solicita a exclusão da conta.
3. O sistema valida a ação (solicita confirmação).
4. O sistema atualiza ou remove os dados do banco.

## Fluxos alternativos
- Senha incorreta para exclusão: nega a operação.

## Pós-condições
- Dados atualizados ou conta removida.
