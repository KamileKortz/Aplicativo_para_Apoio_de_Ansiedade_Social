# Caso de Uso – RF-09: Identificação de padrões

## Descrição
Identificar padrões de ansiedade

## Atores
- Sistema

## Pré-condições
- Dados de ansiedade (RF-05) suficientes para análise.

## Fluxo principal
1. O sistema processa os registros de ansiedade acumulados.
2. O sistema identifica tendências (ex: aumento de ansiedade à noite).
3. O sistema gera uma síntese dos padrões encontrados.
4. O sistema disponibiliza essa informação na área do responsável.

## Pós-condições
- Padrões gerados e prontos para visualização.
