# Caso de Uso – RF-04: Atividades progressivas

## Descrição
Visualizar atividades progressivas.

## Atores
- Criança

## Pré-condições
- Usuário criança cadastrado.

## Fluxo principal
1. O ator acessa a seção de atividades.
2. O sistema verifica o último nível de ansiedade registrado.
3. O sistema filtra as atividades adequadas para aquele nível.
4. O sistema exibe as atividades.

## Fluxos alternativos


## Pós-condições
- Atividades exibidas conforme o progresso da criança.
