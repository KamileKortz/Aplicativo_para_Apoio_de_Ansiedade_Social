# Caso de Uso – RF-16: Conclusão de atividade

## Descrição
Marcar atividades como concluídas.

## Atores
- Criança.

## Pré-condições
- Atividade iniciada.

## Fluxo principal
1. O ator finaliza as etapas da atividade.
2. O sistema marca a atividade como "Concluída".
3. O sistema envia a confirmação para o histórico.

## Pós-condições
- Status da atividade alterado para concluído.
