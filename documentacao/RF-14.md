# Caso de Uso – RF-14: Notificações de registro

## Descrição
Lembretes para registros diários.

## Atores
- Sistema.

## Pré-condições
- Permissão de notificações habilitada.

## Fluxo principal
1. O sistema identifica se houve registro no diário no dia vigente.
2. Caso negativo, o sistema dispara um lembrete.

## Pós-condições
- Lembrete de registro diário enviado.
