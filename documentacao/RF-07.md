# Caso de Uso – RF-07: Diário emocional

## Descrição
Criação de registros descritivos

## Atores
- Criança

## Pré-condições
- Usuário criaça cadastrado.

## Fluxo principal
1. O ator acessa a opçaõ de "Diário".
2. O ator cria uma nova entrada de texto ou seleciona ícones.
3. O ator salva a nota.
4. O sistema armazena com data e hora automática.

## Fluxos alternativos
- Cancelar edição: dados não são salvos.

## Pós-condições
- Registro adicionado ao diário.
