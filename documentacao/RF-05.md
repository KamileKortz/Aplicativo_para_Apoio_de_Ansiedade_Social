# Caso de Uso – RF-05: Registro de ansiedade

## Descrição
O sistema registra ansiedade antes e depois das atividades

## Atores
- Criança

## Pré-condições
- Usuário criança cadastrado.
- Atividade selecionada.

## Fluxo principal
1. O sistema solicita que o ator selecione seu nível de ansiedade atual.
2. O ator seleciona seu nível de ansiedade atual.
3. Após a atividade, o sistema solicita o novo nível de ansiedade.
4. O ator registra o novo nível de ansiedade.
5. O sistema salva os dados comparativos.

## Fluxos alternativos
- Usuário pula o registro: sistema alerta.

## Pós-condições
- Níveis de ansiedade salvos no banco de dados.
