# Caso de Uso – RF-10: Visualização de evolução

## Descrição
Gerar gráficos de evolução.

## Atores
- Usuário

## Pré-condições
-Responsável logado

## Fluxo principal
1. O ator acessa o menu "Evolução".
2. O sistema gera gráficos de linha ou barras com os níveis de ansiedade.
3. O sistema exibe o gráfico na tela.

## Fluxos alternativos
- Dados insuficientes para gerar gráfico: exibe mensagem de erro.

## Pós-condições
- Gráfico exibido visualmente.
