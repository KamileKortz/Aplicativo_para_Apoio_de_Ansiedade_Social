# Caso de Uso – RF-08: Busca no diário

## Descrição
Buscar registros por data, título ou palavras-chave.

## Atores
- Usuário

## Pré-condições
- Existência de registros no diário.

## Fluxo principal
1. O ator acessa a barra de pesquisa do diário.
2. O ator insere uma palavra-chave, título ou seleciona uma data.
3. O sistema filtra os registros.
4. O sistema exibe os resultados correspondentes.

## Fluxos alternativos
- Nenhum resultado encontrado: exibe mensagem "Não encontrado".

## Pós-condições
- Resultados da busca exibidos.
